# Guará Examples Index

# Development usage

## modeling

This section focuses on Domain-Driven Design concepts using Guará. It includes examples of Ubiquitous Language and shows how to express business rules and domain behavior through transactions and assertions, turning tests into executable business models.

## prototyping

This module demonstrates how Guará can be used for rapid prototyping. It shows how to quickly model flows, validate ideas, and iterate on system behavior without needing a full implementation, using transactions as lightweight building blocks.

## rest_api

This example shows how to use Guará for API testing. It demonstrates how to structure API calls as transactions and validate responses using assertions, enabling clear and maintainable test scenarios for REST services.

# Test Automation usage

## crawler

This module demonstrates how to use Guará to automate data extraction workflows. It focuses on navigating systems, collecting structured information, and validating extracted results using transactions and assertions. It shows how Guará can be used beyond testing, acting as an automation tool for crawling and processing data.

## linux_desktop

This example shows how to apply Guará for Linux desktop automation. It demonstrates how transactions can interact with desktop applications, execute actions, and validate outcomes. The focus is on simplifying automation flows using the same DSL used for testing.

## mobile/appium

This module presents how Guará integrates with Appium for mobile automation. It illustrates how to structure mobile test scenarios using transactions and assertions, keeping the same readable flow while interacting with mobile elements and validating behavior.


## unit_test

This module illustrates how Guará can be used for unit testing. It focuses on validating isolated logic using transactions and assertions, providing a consistent approach even at the lowest level of testing.

## web_ui

This section demonstrates web UI automation using Guará. It shows how to interact with web applications, execute user flows, and validate UI behavior. It also reflects the evolution of the framework by aligning assertions with clearer method naming.

## windows_desktop

This module presents how Guará can be applied to Windows desktop automation. It demonstrates interacting with native applications, executing user actions, and validating results using the same transaction-based approach used across other environments.
